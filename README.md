# Granite LLM Security Evaluation Coursework

This repository contains the coursework notebook for implementing system prompt extraction, private file extraction, and guardrail techniques by interacting with the IBM Granite (granite-4.0-h-1b) model via a remote vLLM server.
The main implementation templates is provided in granite_test.ipynb.

## Environment Requirements

- **Python**: 3.12

### Python Dependencies

The notebook relies on the following libraries:

- `transformers`
- `requests`
- `dataclasses`
- `typing`
- `json`, `re`, `os`
- `textwrap`
